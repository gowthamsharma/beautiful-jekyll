The strategy monitors performance of two historically correlated securities. When the correlation between the two securities temporarily weakens(spread increase), i.e. one stock moves up while the other moves down, the pairs trade would be to short the outperforming stock and to long the underperforming one, betting that the "spread" between the two would eventually converge.[3] The divergence within a pair can be caused by temporary supply/demand changes, large buy/sell orders for one security, reaction for important news about one of the companies, and so on.

Pairs trading strategy demands good position sizing, market timing, and decision making skill. Although the strategy does not have much downside risk, there is a scarcity of opportunities, and, for profiting, the trader must be one of the first to capitalize on the opportunity.

rolling Z score transformation¶

def zscore(spread, window):
    rollingwindow = spread.rolling(window=window)
    mean = rollingwindow.mean()
    std = rollingwindow.std()
    z = (spread-mean)/std
    return z

The moving Z-score for a data point x(t) is simply the value of x(t) standardized by subtracting the moving mean and dividing by the moving standard deviation .

pairs trading apple microsoft algorithm¶

import numpy

def initialize(context):

context.appl = sid(24) # appl
context.msft = sid(5061) # msft


context.long_on_spread  = False
context.short_on_spread = False

schedule_function(pairs, date_rules.every_day(), time_rules.market_close(minutes=60))

def pairs(context, data):

aapl = context.appl
msft = context.msft

# rolling window of prices for last 30 days 
prices = data.history([aapl, msft],'price', 30 ,'1d')

# current spread
# spread = data.current(aapl, 'price') - data.current(msft,'price')
current_prices = prices.iloc[-1:]
current_spread = (current_prices[aapl]-current_prices[msft]).mean()
# Z transformation of the spread

rollingspread = prices[aapl]-prices[msft]   
mean = rollingspread.mean()
std = rollingspread.std()  # std of spread over 30 days


if std>0:
    z = (current_spread-mean)/std

    if z>2 and not context.short_on_spread: # ztransform(appl-msft)>2 -> short appl long msft
        order_target_percent(aapl,-0.5)
        order_target_percent(msft, 0.5)
        context.short_on_spread = True
        context.long_on_spread = False

    elif z<-2 and not context.long_on_spread: # ztransform(appl-msft)<-2 -> long appl short msft   
        order_target_percent(aapl,0.5)
        order_target_percent(msft,-0.5)
        context.short_on_spread = False
        context.long_on_spread = True

    elif abs(z)< 0.1:
        order_target_percent(aapl,0) # exit trade
        order_target_percent(msft,0) # exit trade
        context.short_on_spread = False
        context.long_on_spread = False

record(score = z)
record(aapl = data.current(aapl,'price'), msft = data.current(msft,'price'))


# notes : for pairs trading to work we should have low priced volatile stocks ; if the difference is high e.g. aapl 123.95 msft 19.03 in this case the noise is more compared to signal for z transformation
